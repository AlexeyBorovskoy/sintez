# Следующие действия для тестирования

**Дата:** 2026-02-03

---

## Текущее состояние

**Вкладка "Управление контроллером":**
- Протокол: **UTMC** (выпадающий список)
- Версия протокола: **v1,v2c**
- SNMP-уведомления: **INFORM** на **127.0.0.1:10162**

---

## Действие 1: Проверка выпадающего списка протоколов

**Инструкция для пользователя:**
1. На вкладке "Управление контроллером" найдите поле **"Протокол:"**
2. **Нажмите на выпадающий список** (стрелка вниз рядом с "UTMC")
3. Посмотрите, какие протоколы доступны в списке
4. **Сообщите, есть ли там протокол "UG405"**

**Ожидаемый результат:** 
- Список протоколов с возможными вариантами
- Если есть UG405 - это протокол для режима "Удаленный" (UTC Control)

---

## Действие 2: Проверка подвкладки "Планировщик удаленного управления"

**Инструкция для пользователя:**
1. На вкладке "Управление контроллером" найдите подвкладки:
   - "Управление контроллером" (текущая)
   - **"Планировщик удаленного управления"**
2. **Нажмите на подвкладку "Планировщик удаленного управления"**
3. Посмотрите, что там отображается
4. **Сообщите, какие настройки или опции там есть**

**Ожидаемый результат:**
- Возможно, настройки для режима "Удаленный"
- Возможно, настройки протокола UG405 и SCN
- Возможно, планировщик для удаленного управления

---

## Важные замечания

1. **Текущий протокол UTMC** - это протокол для режима Monitor
2. **Для режима UTC Control** нужен протокол **UG405** (согласно документации)
3. **SCN** нужно задать при выборе протокола UG405 (формат: COX, где X - число)

---

## Следующий шаг

**Выполните Действие 1:** Проверьте выпадающий список протоколов и сообщите, есть ли там UG405.

После этого мы сможем настроить протокол UG405 и перейти в режим "Удаленный" (UTC Control) для дальнейшего тестирования.
