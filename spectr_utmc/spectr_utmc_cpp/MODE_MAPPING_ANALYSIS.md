**Наблюдение:** 
- Веб-интерфейс показывает изменение режима ("Фиксированный" → "Адаптивный")
- SNMP все еще показывает `INTEGER: 2` (Monitor)

**Возможные объяснения:**

**Вариант A:** Оба режима соответствуют Monitor (2) в SNMP
- "Фиксированный" = Monitor (2) с фиксированным планом
- "Адаптивный" = Monitor (2) с адаптивным планом
- Разница только в плане работы (План 1 vs План 2), а не в режиме SNMP

**Вариант B:** Задержка обновления SNMP
- Контроллер изменил режим, но SNMP еще не обновился
- Нужно подождать несколько секунд

**Вариант C:** "Адаптивный" соответствует другому значению
- Возможно, "Адаптивный" = Standalone (1) или UTC Control (3)
- Но контроллер еще не применил изменение через SNMP

### 2. Фаза

**Наблюдение:**
- Фаза изменилась: 10/10 → 5/12 ✅
- Битовая маска изменилась: `02 00` → `80` ✅

**Вывод:** Контроллер действительно изменил свое состояние, фаза обновилась.
