# Анализ вкладки "Управление контроллером"

**Дата:** 2026-02-03

---

## Текущие настройки

### Протокол
- **Протокол:** UTMC (выпадающий список)
- **Версия протокола:** v1,v2c (выпадающий список)

### SNMP-уведомления
- **Тип уведомления:** INFORM
- **IP-адрес получателя:** 127.0.0.1
- **Порт получателя:** 10162

---

## Наблюдения

1. **Протокол UTMC выбран** - это текущий протокол управления
2. **SNMP-уведомления настроены** на отправку INFORM на 127.0.0.1:10162
3. **Есть подвкладка:** "Планировщик удаленного управления" - возможно, там настраивается режим "Удаленный"

---

## Что нужно проверить

### 1. Проверка выпадающего списка протоколов
- **Действие:** Нажать на выпадающий список "Протокол" (рядом с UTMC)
- **Цель:** Проверить, есть ли в списке протокол "UG405"
- **Ожидаемый результат:** Список протоколов с возможными вариантами

### 2. Проверка подвкладки "Планировщик удаленного управления"
- **Действие:** Перейти на подвкладку "Планировщик удаленного управления"
- **Цель:** Проверить, есть ли там настройки для режима "Удаленный" или UTC Control
- **Ожидаемый результат:** Настройки для удаленного управления

---

## Следующие шаги

1. Проверить выпадающий список протоколов на наличие UG405
2. Проверить подвкладку "Планировщик удаленного управления"
3. Если UG405 есть - настроить его с SCN
4. Если режим "Удаленный" есть - переключиться на него
