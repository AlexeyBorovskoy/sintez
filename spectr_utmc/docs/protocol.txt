Для управления контроллером по протоколу UG405 необходимо выполнить следующие настройки в web интерфейсе:

Во вкладке «Управление контроллером» выбрать из списка протоколов UG405 и задать SCN.

Обязательным требованием является следующий формат: SCN = COX, где "CO" обязательные символы латинскими буквами, а X - число от 1 до 9999;

После проведенных настроек контроллер готов к работе с протоколом UG 405 с использованием следующих OID:

 

GetTime: // получение системного времени

    """1.3.6.1.4.1.13267.3.2.3.2.0 YYYYMMDDHHmmssZ

    This object returns the time the configuration data (utcCoreConfig) available via the MIB was last changed,

    as YYYYMMDDHHmmssZ. Z indicates zulu or GMT

   

    This format is based upon ExtUTCTime from RFC:2578 and is the  format used in other SNMP and UTMC timestamps.

    STRING: '19700101000000Z'"""

 

GetVersion: // получение версии ПО

    """1.3.6.1.4.1.13267.3.2.1.2

    String For Application Version.   Format is vendor specific.

    STRING: 'V001'"""

 

SetGetMode: // установка/получение  состояния работы 1-автономный, 2-сервисный, 3-удаленный

    """1.3.6.1.4.1.13267.3.2.4.1.0"""

 

GetPhase: // получить текущий номер фазы

    """1.3.6.1.4.1.13267.3.2.5.1.1.3"""

 

SetPhase : // установить фазу

    """1.3.6.1.4.1.13267.3.2.4.2.1.5"""

 

StageDemandBits : // возвращает текущую фазу после передачи информации о фазе в установленном формате

    """1.3.6.1.4.1.13267.3.2.4.2.1.4"""

 

SetPhaseTrue : // разрешает установку фазы

    """1.3.6.1.4.1.13267.3.2.4.2.1.15"""

 

GetError : // получить ошибки

    """1.3.6.1.4.1.13267.3.2.5.1.1.5

    Condition 1 confirms that the detector fault monitor system indicates a detector failure.

    INTEGER: 0"""

 

GetWarnings : // получить предупреждения

    """1.3.6.1.4.1.13267.3.2.5.1.1.16

    Condition 1 confirms that an entry  is in the system fault log.

    INTEGER: 1"""

 

GetRCP : // получить состояние панели внешнего управления (актуально для Москвы)

    """1.3.6.1.4.1.13267.3.2.5.1.1.15

    Condition 1 confirms that Manual Control is either in operation

    or requested as specified in an associated Works Specification.

    INTEGER: 1"""

 

GetAF : // получение  состояния ЖМ

    """1.3.6.1.4.1.13267.3.2.5.1.1.36

    Condition '1' confirms that the controller signals are in flashing amber mode.

    This bit is only specified for export (non-UK) applications.

    INTEGER: 1"""

 

SetAF : // включение  состояния ЖМ

    """1.3.6.1.4.1.13267.3.2.4.2.1.20

    Where a condition '1' exists for a minimum of 10 seconds,

    the signals shall be set to flashing amber during

    a nominated stage provided that all minimum running periods have expired.

    Where a condition '0' exists for a minimum of 10 seconds,

    the signals shall switch on in accordance with the start up sequence.

    This facility is only specified for export (non-UK) applications.

    INTEGER: 1"""

 

SetOFF : // включение режима «Все выключено»

    """1.3.6.1.4.1.13267.3.2.4.2.1.11

    Where a condition 1 exists for a  minimum of 10 seconds, the signals shall switch on in

    accordance with the Start Up  Sequence.. Where a condition 0 is  present for a minimum of 10

    seconds, the signals shall switch off during a nominated stage, provided that all minimum running periods have expired."""

 

GetOFF : // получение состояния «Все выключено»

    """1.3.6.1.4.1.13267.3.2.5.1.1.45"""

 

GetAdaptive : // получение  состояния адаптивного управления

    """1.3.6.1.4.1.13267.3.2.5.1.1.46"""

 

OID с именами 'SysDescr', 'Uptime', 'GetVersion', 'GetPhase', 'GetError', 'GetWarnings', 'GetRCP', 'GetAF', 'SetPhase', 'StageDemandBits', 'SetPhaseTrue', 'SetAF', 'SetOFF', 'GetOFF' работают с передачей SCN

 

OID с именами  'SetGetMode', 'GetTime' работают без передачи SCN.

 


 
 
В продолжение тестирования вашего ДК в Санкт-Петербурге мы готовы попробовать его интегрировать в состав АСУДД «Спектр» по протоколу UG405 (как ранее были интегрированы контроллеры  EC-2)
Для этого вам необходимо сообщить, каким образом можно выполнить конфигурирование протокола UG405 на вашем оборудовании, так как представленные конфигураторы это сделать не позволяют.
