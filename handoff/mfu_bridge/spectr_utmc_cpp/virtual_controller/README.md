# Виртуальный контроллер SINTEZ UTMC

Виртуальный контроллер для тестирования без риска для реального оборудования.

## Установка

```bash
pip3 install pysnmp
```

## Запуск

```bash
python3 snmp_agent_emulator.py [path_to_config.json]
```

Если путь к конфигурации не указан, будет использована конфигурация из снэпшота:
`/home/alexey/shared_vm/spectr_utmc/spectr_utmc/controller_snapshot/home/voicelink/rtc/resident/config`

## Функции

- Эмуляция режимов работы (Local, Standalone, Monitor, UTC Control)
- Эмуляция фаз и их временных параметров
- Эмуляция защиты от ошибок мигания
- Эмуляция минимальных периодов работы фаз
- Эмуляция специальных фаз (nominated stages)
- Логирование всех операций

## Тестирование

Используйте стандартные SNMP инструменты для тестирования:

```bash
# Проверка режима работы
snmpget -v2c -c UTMC localhost 1.3.6.1.4.1.13267.3.2.4.1

# Активация мигания
snmpset -v2c -c UTMC localhost \
  1.3.6.1.4.1.13267.3.2.4.1 i 3 \
  1.3.6.1.4.1.13267.3.2.4.2.1.20 i 1
```

## Статус

⚠️ **В РАЗРАБОТКЕ** - базовый эмулятор создан, но требует интеграции с SNMP сервером для полной функциональности.
